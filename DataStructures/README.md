# Data Structures

This folder contains implementations of fundamental Data Structure concepts and algorithms using C.

## Topics Covered
- Stack Operations
- Linked List
- Trees
- Graph Traversal
- Searching Algorithms
- Sorting Algorithms
- Hashing
- Polynomial Operations
- Expression Evaluation

## Algorithms Included
- BFS & DFS
- Bubble Sort
- Insertion Sort
- Linear Search
- BST Operations
- Infix to Postfix Conversion
- Postfix Evaluation

## Purpose
Practicing problem-solving and understanding the implementation of core Data Structure concepts.
